# Installation
1. Clone the repo
 ```bash
   git clone https://github.com/pesto-students/meditationmitra-fe-n13-alpha-1.git
 ```
2. Install NPM packages
 ```bash
   cd meditationmitra-fe-n13-alpha-1 && npm install
 ```
3. Run
 ```bash
    npm start
 ```
4. Open http://localhost:3000 to view it in the browser
